# UrbanHive — Intelligent City Mobility OS

UrbanHive is a premium, real-time City Mobility Operating System designed to orchestrate and optimize urban infrastructure. Inspired by Apple's signature design guidelines, it features clean grid alignments, Outfit typography, responsive glassmorphic cards, and detailed visualization panels.

## 🚀 Key Features

- **Dynamic Command Center**: A unified overview dashboard including real-time KPI metrics, active system alerts, and comprehensive corridor analytics.
- **Interactive GIS Map Engine**: Integrates React-Leaflet with dynamic camera controls that smoothly glide (`flyTo`) between key metropolitan regions (Indore, Bhopal, Pune).
- **Intricate Indore Transit Network**: Simulates 16 major Indore squares (Vijay Nagar, Rajwada, Palasia, Bengali Square, etc.) with physical coordinate connections, congestion indicators, and active BRTS routes.
- **EV charging Orchestration**: Live load split donut graphs, queue metrics, and an AI-powered charging rebalance system.
- **Smart Parking Locator**: Color-coded parking occupancy levels and proximity metrics.
- **Real-Time Camera Feeds**: Interactive traffic video feed streams with AI bounding boxes for real-time pedestrian and vehicle count calculations.

## 🛠️ Technology Stack

- **Framework**: React 18, Vite, TypeScript
- **Styling**: Vanilla CSS with Tailwind CSS tokens and Outfit fonts
- **Icons**: Lucide React
- **Visuals & Charts**: Recharts (Pie, Radial Bar, Area, and Bar charts)
- **Map Library**: React-Leaflet & Leaflet.js
- **Animations**: Framer Motion
